# 关于本站

本站收录 GitHub 上 Star 数最多的 500 个 AI 开源项目，为中文开发者提供：

- 一句话中文简介
- 3-5 个具体适用场景
- 上手难度评估
- 同类项目替代推荐

数据每周自动更新，所有信息可追溯到原始仓库。

## 内容来源

- 仓库元数据来自 GitHub Search API
- 中文解读由大语言模型基于 README 生成
